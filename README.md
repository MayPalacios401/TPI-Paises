* Mayra Palacios

## Descripción

Este proyecto fue desarrollado para la materia Programación I de la Tecnicatura Universitaria en Programación (UTN).

El sistema permite gestionar información de distintos países almacenados en un archivo CSV, ofreciendo funcionalidades de:

* Agregar países.
* Actualizar información.
* Buscar países.
* Filtrar por continente.
* Filtrar por rango de población.
* Filtrar por rango de superficie.
* Ordenar registros.
* Obtener estadísticas generales.

## Tecnologías utilizadas

* Python 3
* CSV
* GitHub

## Ejecución

1. Descargar los archivos del repositorio.
2. Abrir una terminal en la carpeta del proyecto.
3. Ejecutar:

```bash
python main.py
```

1. Agregar país.

4. Filtrar países.

7. Salir.
